# Contributing

This is mainly a personal lab-solution repository, so large feature contributions are not expected.

Issues and small corrections are welcome, especially for:

- broken file paths;
- unclear README descriptions;
- typos in comments or documentation;
- simulation instructions that do not work.

Please do not submit copied official solutions or private course materials that should not be redistributed.

## Commit style

Use short and descriptive commit messages, for example:

```text
docs: improve Lab8 usage notes
fix: correct ALU test vector path
chore: ignore Vivado generated files
```
